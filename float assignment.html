CSS Float and Display Assignment

Understanding how CSS positioning works

Part 1: Two Images Side by Side Using Float

How it works: The images float side by side because we use float left on both images. Each image takes up 45% of the width with some margin space between them.
img { float: left; width: 45%; margin: 10px; }

Part 2: Why Float Does Not Work With Display None

The Simple Answer

When you write display none on an element, it completely removes that element from the page. It is like the element does not exist at all.

Since float only works on elements that are actually shown on the page, there is nothing for float to affect when display is set to none.

Think of it this way:

  • Float left says: show this element on the left side
  • Display none says: do not show this element at all
  • Display none always wins because you cannot position something that does not exist
Pro Tip: If you want to hide an element but keep it in the layout so float still works, use visibility hidden instead of display none.
.hidden { display: none; float: left; /* Float has NO effect here */ } .invisible { visibility: hidden; float: left; /* Float WORKS here */ }
